# v1.5.0

## Highlights

- Added a full Windows stock dashboard and single-stock workbench.
- Integrated bull stock calculator strategy ideas shared by [Atanvardo_1](https://bbs.nga.cn/nuke.php?func=ucp&uid=8096803), 娜美, Xhox, and other NGA contributors.
- Added watchlist, focus-watch, position board, custom groups, code/name search, fuzzy search, CSV import/export, clear list, and drag ordering.
- Added 3-second quote refreshes, market strip support for Shanghai, Shenzhen, ChiNext, and STAR 50, plus yellow/white intraday index charts.
- Added single-stock intraday/K-line charts with volume, MA/BOLL overlays, swing high/low, Fibonacci retracement, intraday pressure/support, position profit and loss, and bull strategy signals.
- Auto-fills swing high/low from daily K-line and MACD swing logic when possible, while keeping manual edits available.
- Exposes watchlist, focus-watch, and position context to the local AI analysis flow so the AI side can reason over the same stock lists.

## Notes

The stock strategy output is for observation, review, and risk discussion only. It is not investment advice and the app never trades automatically.
